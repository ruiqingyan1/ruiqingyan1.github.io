1、pandas数据结构

两个常用的工具数据结构：Series、DataFrame

1.1 Series

一维的数组型对象，索引index + 值序列

形式类似字典，可以直接传入字典生成Series

1.2 DataFrame

矩阵的数据表，包含已排序的列序列，既有行索引，又有列索引（一个共享相同索引的Series的字典）

frame.values、frame.columns、frame

有效输入 类型 注释
2D ndarry 数据的矩阵，行和列的标签是可选参数
数组、列表和元组构成的字典 每个序列为DataFrame的一列，所有序列长度相等
NumPy结构化/记录化数组 与数组构成的字典一致
Series构成的字典 每个值为一列
字典或Series构成的列表 字典键或Series索引联合起来形成Data的列标签
列表或元组构成的列表 与2D ndarray一致
其他DataFrame 若不显示传递索引，则用原索引
NumPy MaskedArray 与2D ndarray相似，隐蔽值显示成NaN

1.3 索引对象

index的属性和方法

方法 说明
append 连接另一个Index对象，产生一个新的Index
difference 计算差集，并得到一个Index
intersection 计算交集
union 计算并集
isin 计算一个指示各值是否都包含在参数集合中的布尔型数组
delete 删除索引处的元素，并得到新的Index
drop 删除传入的值，并得到新的Index
insert 将元素插入到索引处，并得到新的Index
is_monotonic 当各元素均大于等于前一个元素时，返回True
is_unique 当Index没有重复值时，返回True
unique 计算Index中唯一值的数

2、 基本功能

2.1 reindex重建索引

obj.reindex([1,2,3,4,5]) 等价于 obj.index = range(1,6)

Obj.reindex(range(1,11),method=‘ffile’) , 这里的ffile方法会向前填充

参数 说明
index 用作索引的新序列。既可以是Index实例，也可以是其他序列型的Python数据结构。Index会被完全使用，就像没有任何复制一样
method 插值（填充）方式，具体参数请参见下表
fill_value 在重新索引的过程中，需要引入缺失值时使用的替代值
limit 前向或后向填充时的最大填充量
tolerance 向前后向后填充时，填充不准确匹配项的最大间距（绝对值距离）
level 在Multilndex的指定级别上匹配简单索引，否则选取其子集
сору 默认为True，无论如何都复制；如果为False，则新旧相等就不复

2.2 轴向上删除条目

obj.drop( [标签] [,axis = ] [,inplace = True] ) #删除标签对应行，axis默认0表示行，若设为1则表示删除标签对应列，inplace表示原对象不返回新对象

2.3 使用loc和iloc选择数据

loc 在索引的时候会去到边缘，正常切片需要上限+1

iloc 类似正常索引切片，但是是先行后列

正常切片都是先选列再选行

类型 说明
df[val] 从DataFrame选取单列或一组列
df. loc [val] 通过标签，选取 DataFrame 的单个行或一组行
df. loc [ : , val] 通过标签，选取单列或列子集
df. loc [val1, val2] 通过标签，同时选取行和列
df. iloc [where] 通过警数位置，从DataFrame 选取单个行或行子集
df. iloc [ : , where] 通过整数位置，从 DataFrame 选取单个列或列子集
df. iloc[ where_i , where_j ] 通过整数位置，同时选取行和列
df. at [label_i, label_j ] 通过行和列标签，选取单一的标量
df. iat [ i , j ] 通过行和列的位置（整数），选取单一的标量
reindex方法 通过标签选取行或列
get_value, set_value方法 通过行和列标签选取单一值

方法 说明
add, radd 用于加法（+）的方法

div,rdiv 用于除法（/）的方法
floordiv, rfloordiv 用于底除（/）的方法
mul, rmul 用于乘法（*）的方法
pow, rpow 用于指数（**）的方
