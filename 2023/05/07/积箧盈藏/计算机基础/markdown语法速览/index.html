markdown语法速览

发表于2023-05-07|更新于2023-05-07|计算机基础markdown
|字数总计:377|阅读时长:1分钟|阅读量:

主要：

样式 符号 展示 加
标题 #
斜体 *这是斜体* 这是斜体
粗体 **这是粗体** 这是粗体
粗斜体 ***这是粗斜体* 这是粗斜体
小字体 ~这是小字~ ~这是小字~
删除线 ~~这是删除线~~ 这是删除线> 空格
下划线 这是下划线 分割线（Horizontal Rule） 这是下划线—- 回车
无序列表 - 空格
有序列表 1. 2. 3. 空格
链接 [百度](www.baidu,com) 百度
图片 ![ 名字 ](图片地址\链接)
引用 > 空格
分割线 —- 回车
目录 [toc] 回车
代办列表 -[ ] / - [x] 空格

  • 这里输入 |a|b| 然后回车 就可以生成表格
  • 如果有需要合并的单元格，请用html内置标签
