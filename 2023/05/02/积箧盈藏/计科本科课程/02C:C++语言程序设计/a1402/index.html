1、数据类型

关键字：一些预先定义的

常量：程序中值不变化的量

变量：程序中值可以变化的量

注：

  1. int定义整型变量，float定义浮点型变量，char定义字符变量。
  2. 常量在程序执行过程中不可用修改。
  3. 变量命名只能有数字、字母、下划线组成。
  4. int4字节，float4字节，char1字节。
  5. “hello”六字节，最后一个是‘\0’

2、类型转换/进制转换

类型转换分显性类型转换和隐形类型转换，靠经验去判断就好，(float)a这种属于显性。

十进制0~9 八进制0~7 十六进制0~9和A-F，具体表示十进制直接写 八进制前加0,例如021=17 十六进制前加0x,例如0x7c=124

进制转换就是正常计算，没什么好说的
